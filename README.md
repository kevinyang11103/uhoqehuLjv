# 家教管理小程序 python1274

本项目是一款基于Springboot、MySQL和nodejs的家教管理小程序，包含用户端和管理端，致力于提供便捷的家教服务管理。

## 技术栈

### 开发环境
- 操作系统：Windows 10/11
- 数据库：MySQL 8.0
- Python版本：Python 3.8+

### 后端技术栈
- Web框架：Django 4.0
- 数据库ORM：Django ORM
- API开发：Django REST framework
- 权限管理：Django Auth

### 前端技术栈
- HTML5
- CSS3
- JavaScript
- Bootstrap 4
- jQuery

### 数据库工具
- MySQL Workbench
- Navicat Premium

## 功能模块

本项目主要包含以下功能模块：
- 用户注册与登录
- 家教信息发布与管理
- 课程预约与取消
- 评价与反馈
- 数据分析与统计

## 系统角色
- 普通用户：可以注册、登录、发布家教需求、预约家教、评价家教
- 家教老师：可以注册、登录、发布家教信息、接受课程预约
- 管理员：负责管理用户、家教信息、课程预约等，以及查看系统数据统计

## 运行环境
- 后端：基于Python 3.8+的Django框架
- 前端：HTML5、CSS3、JavaScript
- 数据库：MySQL 8.0

## 部署步骤
1. 安装Python 3.8+
2. 安装MySQL 8.0
3. 克隆项目代码
4. 安装依赖库
5. 数据库迁移与初始化
6. 运行项目
